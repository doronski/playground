# Repository Status

## What this package is
This is a **GitHub-repo-ready package** for a dedicated political strategists project.

## What it contains now
- the ranked list
- the artefacts index
- source registries
- archive guide
- recommended works manifest
- public-domain text copies already bundled

## What it does not contain yet
- full-text copies of copyrighted commercial books
- licensed database exports
- anything that would require infringing reproduction

## How to complete it after purchase
1. Buy or otherwise lawfully obtain the recommended books.
2. Upload the files you want indexed.
3. Place them under `acquisitions/purchased/` or a similar folder.
4. Update `data/recommended_works_manifest.csv` from `purchase_recommended` to `user_supplied_copy`.
