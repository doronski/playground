# Political Strategists Repository

This repository package contains a ranked list of 100 major political strategists, an artefact index, and a small set of downloaded open texts.

## Contents

- `data/top_100_political_strategists.csv`
- `data/top_100_political_strategists.json`
- `data/artefacts_index.csv`
- `docs/methodology.md`
- `docs/top_100_ranked.md`
- `sources/source_registry.csv`
- `sources/archive_sources.csv`
- `downloads/`

## What is inside the artefacts index

The artefacts index contains:
- exact links where an open primary text was located
- discovery links for books and scanned editions
- archive discovery links for the strategist as a person
- source metadata and notes

Total indexed artefact rows: **1,038**

## Downloaded local copies

A limited subset of openly accessible texts is stored in `downloads/`.
These are included because they are public domain or openly accessible archival pages.
